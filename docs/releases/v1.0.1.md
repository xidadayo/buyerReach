# BuyerReach v1.0.1

Release date: 2026-07-22

This patch preserves the immutable `v1.0.0` baseline and makes NAS upgrades
independent of executable file modes. `upgrade.sh` now invokes `backup.sh`
through POSIX `sh`, and the deployment examples do the same for `upgrade.sh`.

There are no application schema, Pipeline, Prompt, Policy, Adapter, or API
changes from `v1.0.0`. Database head remains `20260721_0030`.

The verification status and known baseline failures recorded in
`docs/releases/v1.0.0.md` remain applicable.
