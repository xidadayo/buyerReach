## Deployment audit

Use `scripts/upgrade.sh` for subsequent deployments. It writes append-only
JSONL records containing old/new commits, requested version, database revisions,
backup path, outcome, and rollback status. Store `AUDIT_DIR` on persistent NAS
storage and retain it with off-device backups.
