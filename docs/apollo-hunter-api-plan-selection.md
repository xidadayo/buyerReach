# BuyerReach Apollo / Hunter 套餐价格对比

> 价格核对日期：2026-07-22。按中国银行美元现汇卖出价约 1 美元 = 6.79 元人民币估算。以下人民币价格均为约数，不含税费、信用卡手续费和汇率波动。

## 1. 计算前提

- 每月处理 100、300 或 500 家公司；
- 每家公司最多获取 5 位联系人及工作邮箱；
- 不获取手机号；
- 年付方案按相同用量连续运行 12 个月计算；
- Apollo + Hunter 表示两条管线都完整运行，因此需要同时购买两家套餐。

对应的最大联系人处理量：

| 100 家 | 500 人 | 6,000 人 |
| 300 家 | 1,500 人 | 18,000 人 |
| 500 家 | 2,500 人 | 30,000 人 |

## 4. 套餐额度说明

| Vendor 套餐 | 公开额度 | 适用档位 |
|---|---:|---|
| Apollo Basic | 30,000 credits/席/年 | 每月 100 或 300 家 |
| Apollo Professional | 48,000 credits/席/年 | 每月 500 家，保留公司搜索和运行余量 |
| Hunter Starter | 2,000 credits/月或 24,000/年 | 每月 100 或 300 家 |
| Hunter Growth | 10,000 credits/月或 120,000/年 | 每月 500 家 |

500 家公司按每家公司 5 位联系人计算，全年最多需要处理 30,000 位联系人。Apollo Basic 的 30,000 credits 没有为公司搜索、分页和运行波动留下余量，因此选择 Professional。Hunter Starter 每月只有 2,000 credits，无法覆盖最多 2,500 位联系人，因此选择 Growth。

| 每月公司数 | 控制成本 | 优先 Apollo | 双来源覆盖 |

| 100 家 | Hunter Starter：约 ¥333/月或 ¥2,770/年 | Apollo Basic：约 ¥401/月或 ¥3,993/年 | 约 ¥733/月或 ¥6,763/年 |
| 300 家 | Hunter Starter：约 ¥333/月或 ¥2,770/年 | Apollo Basic：约 ¥401/月或 ¥3,993/年 | 约 ¥733/月或 ¥6,763/年 |
| 500 家 | Apollo Professional：约 ¥672/月或 ¥6,437/年 | Apollo Professional：约 ¥672/月或 ¥6,437/年 | 约 ¥1,684/月或 ¥14,911/年 |

如果只是试运行 1 个月，选择月付；如果预计稳定运行 5～7 个月以上，通常年付更划算。正式采购前仍需在账户 checkout 中确认 API 权限、Master API Key 和最终税后价格。
