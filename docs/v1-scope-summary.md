# V1 Scope Summary From PRD

The PRD baseline defines BuyerReach V1.0 as a customer data foundation platform.

## Must Build

- Login, users, roles, basic data permissions
- Search tasks and async background execution
- Company, brand, website, contact, email relationship model
- Website public information parsing
- At least one contact provider, one email finder provider, and one verifier provider
- Email statuses, scoring, email pools, and suppression pool
- Duplicate detection and manual merge workflow
- Source evidence, audit logs, tags, and basic custom fields
- CSV/XLSX import/export
- Feishu webhook notification
- Docker Compose, backup/restore, and migration mechanism

## Reserved Only

- Outreach accounts, templates, sequences, sending, and reply models
- Complete CRM opportunity, quote, sample, and order modules
- AI customer analysis and personalized outreach
- Third-party CRM bidirectional sync
- Brand news/product monitoring
- Mobile app and browser plugin
